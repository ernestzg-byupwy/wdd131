CSS

Role: Styling

History

CSS was proposed in 1994 by Norwegian web pioneer Håkon Wium Lie, also working at CERN. Before CSS, all visual styling was done directly in HTML using attributes like color, bgcolor, and <font> tags — mixing content and presentation in away that made sites difficult to maintain and update.

CSS Level 1 was officially published by the W3C in December 1996. It covered basic properties like fonts, colors, and spacing. CSS2 followed in 1998, adding support for positioning, z-index, and media types.

Browser adoption in the late 1990s was notoriously inconsistent. Developers often had to write duplicate code to handle differences between Netscape Navigator and Internet Explorer, a period sometimes called the "browser wars."

CSS2.1 (published 2011) corrected and clarified many CSS2 behaviors. CSS3, rather than being a single monolithic specification, was broken into independent modules — selectors, box model, colors, animations, flexbox, grid — each developed and updated on its own timeline. This modular approach continues today.

The introduction of Flexbox (around 2012–2015) and CSS Grid (2017) revolutionized web layout, finally giving developers powerful, native tools to create complex, responsive designs without relying on float hacks or JavaScript.

Ways to Apply

Method How
External stylesheet <link rel="stylesheet" href="styles.css"> in <head>
Internal style <style> block inside <head>
Inline style style="color: red" directly on an element

External stylesheets are the recommended approach for maintainability.

Selectors

Selector Example Targets
Type p { } All <p> elements
Class .card { } Elements with class="card"
ID #header { } Element with id="header"
Universal * { } Every element
Descendant nav a { } <a> inside <nav>
Pseudo-class a:hover { } <a> when hovered
