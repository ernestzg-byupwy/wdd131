Quick Web Core



Description:

This name reflects the nature and purpose of the website as a reference on Core Web Technologies.

Site Purpose

The site provides is meant to serve as a quick reference on short history and fundamental concepts of Core Web Technologies like HTML, CSS and JavaScript. Users will be able to find some history about how and when each technologies was born, alongside soe basic information on syntax and code examples.

Scenarios

What is the history of HTML CSS and JavaScript?

The site provides information on the history of core web technologies, like who created it and what was the need they were looking to solve.



Where do I start?

The site provides some basic code examples to help you understand how the web works and provides links for well-known sites where the user can learn more.
